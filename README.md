# 💗 Est-ce que Capucine mérite son ensemble Teveo ?

🔗 **[Voter ici](https://nachoelchihuahua.github.io/suprise-vote/)**

## C'est quoi ce projet ?

Une page web interactive à partager avec n'importe qui. Elle pose une question très importante :

> *Est-ce que Capucine mérite son ensemble Teveo ?*

L'utilisateur a deux choix :

- **OUI** — la bonne réponse. Affiche un message de félicitations.
- **NON** — le bouton s'enfuit dès que la souris s'approche. Il est impossible à cliquer.

## Fonctionnalités

- Cœurs flottants en arrière-plan
- Bouton **NON** qui esquive la souris (et le doigt sur mobile)
- Page de succès après avoir cliqué **OUI**

## Stack

HTML + CSS + JavaScript vanilla — aucune dépendance, aucun serveur.
